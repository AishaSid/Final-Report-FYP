# Final Report - FYP

A professional final year project (FYP) report compiled using LaTeX.

## Overview

This repository contains the complete LaTeX source code and documentation for a Final Year Project report. The project uses industry-standard LaTeX tools for document compilation and formatting.

## Technologies

- **LaTeX** - Document preparation and typesetting
- **latexmk** - Automated LaTeX build tool

## Repository Structure

- `*.tex` - LaTeX document source files
- Configuration files for document compilation

## Getting Started

### Prerequisites
- LaTeX distribution (e.g., TeX Live, MiKTeX)
- latexmk

### Building the Document

To compile the report:

```bash
latexmk main.tex
